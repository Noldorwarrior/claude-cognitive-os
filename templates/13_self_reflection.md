# 13 — Журнал самоаудитов Claude

Журнал самоаудитов по 8 осям после каждой комплексной задачи.
Источник уроков, ошибок, паттернов и калибровки.

**Когда активируется:**
- Триггер от [[productivity:memory-management]]: задача отмечена `[x]`.
- Явная команда `оцени свою работу` / `самоанализ`.
- Автоматический хук от завершения скилла с пометкой `requires_reflection: true`.
- Из скилла [[govdoc-analytics:self-reflection]] (внешнее делегирование).

## 8 осей скоринга

| # | Ось | Что проверяется | Упрощённая эвристика (fallback) |
|---|---|---|---|
| 1 | Корректность | Цифры, даты, имена точны | Сверка с исходником |
| 2 | Полнота | Все пункты запроса выполнены | Чек-лист пунктов |
| 3 | Формат | Оформление по правилам | Сверка с [[01_user_profile]] |
| 4 | Источники | Ссылки на первоисточники | Есть ссылки? |
| 5 | Логика | Отсутствие противоречий | Поиск противоречий (М7) |
| 6 | Аудитория | Тон / глубина под получателя | Моделирование (М27) |
| 7 | Калибровка | Заявл. уверенность ≈ факт. | Гэп ≤ 10% |
| 8 | Уроки | Что извлечено для будущего | Минимум 1 lesson / ec / insight |

## Связь с внешними скиллами

### govdoc-analytics:self-reflection

Если установлен — делегирует туда. Возвращает полноценный отчёт по 8
осям. Лог копируется сюда.

### Fallback: internal_8axis_scoring

Если не установлен — Claude оценивает сам по упрощённой эвристике.
Результат менее глубокий, но достаточный для базовой фиксации.

## Правила записи

1. **Уникальный ID** `sr-NNN`.
2. **Все 8 осей заполнены** (даже если упрощённо).
3. **Связи обязательны** — ссылки на `lesson-*`, `ec-*`, `pat-*`,
   `domain-*`, `wm-*`.
4. **Журнал решений** — основные выборы, сделанные в задаче.
5. **Запись в [[11_confidence_scoring]]** — обязательно для расчёта
   калибровки.

## Агрегирование за период

Раз в месяц через action-скилл `reflect` (или вручную) — построение
отчёта за период:
- Средний скор по 8 осям.
- Топ-3 слабых оси → приоритеты на улучшение.
- Топ-5 уроков / ошибок / паттернов.
- Изменения калибровки.

## Команды

| Команда | Действие |
|---|---|
| `самоанализ` | Запустить `reflect` для текущей задачи |

| `покажи слабые оси` | Где Claude систематически ошибается |
