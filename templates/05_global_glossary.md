# 05 — Сквозной глоссарий

Сквозные термины, аббревиатуры, нейминг, используемые в 2+ проектах
или имеющие значение для всех диалогов с пользователем.

**Что попадает сюда:**
- Аббревиатуры (РОЗ, ДВФМ, ИРИ, РДДМ, РСВ и т.д.).
- Внутренние коды проектов и их расшифровки.
- Нейминг специфических ролей (куратор, модератор, спикер).
- Термины госсектора РФ с контекстом применения.

**Что НЕ попадает:**
- Общеизвестные термины (госорганизация, проект, отчёт).
- Термины одного проекта (→ в локальную карту проекта).
- Люди и организации (→ [[07_global_entities]]).

## Дубль-связь с govdoc-analytics:govdoc-context

Часть аббревиатур может быть уже в [[govdoc-analytics:govdoc-context]]
(справочник госсектора). **Не дублировать.** Если термин уже там —
только ссылка, без повтора определения.

## Правила записи

1. **Сквозное использование** (2+ проекта ИЛИ все диалоги).
2. **Уникальный ID** `term-NNN`.
3. **Не дублирует** govdoc-context.
4. **Категория обязательна.**
5. **Расшифровка + определение** обязательны.
6. **Примеры применения** — минимум 1.

## Fuzzy-matching

Для аббревиатур добавляются синонимы / варианты написания (кириллица /
латиница / с пробелами), чтобы Claude распознавал термин при любой форме.

<!-- Пример:
### term-001 — РОЗ
- Синонимы: Росмолодёжь, Rosmolodezh, РосМолодежь, Росмолодежь
-->

## Команды

| Команда | Действие |
|---|---|
| `покажи глоссарий` | Все активные термины |
| `что такое X` | Поиск по глоссарию + fuzzy match |
| `добавь термин: X = Y` | Новая запись |
