# 07 — Сквозные сущности

Люди, организации, события, инструменты, упоминаемые в **2+ проектах**
(порог `thresholds.cluster_min_projects`). Если упоминается только в
одном — хранится в локальной карте проекта.

**Что попадает сюда:**
- Люди: имя, роль, организация, контекст взаимодействия.
- Организации: название, профиль, роль в проектах пользователя.
- События: форум / конгресс / заседание, упомянутое в 2+ проектах.
- Инструменты / платформы: если используются в 2+ проектах.

## Правила записи

1. **Порог упоминаний:** минимум `thresholds.cluster_min_projects` (default 2)
   разных проекта.
2. **Уникальный ID** `ent-NNN`.
3. **Тип обязателен:** person | organization | event | tool.
4. **Контексты явные** — список проектов с ссылками.
5. **Счётчик упоминаний** — обновляется при каждой новой ссылке.
6. **Для людей:** форма обращения обязательна (для официальных писем
   через [[govdoc-analytics:letter]]).

## Связь с govdoc-context

Часть организаций уже есть в [[govdoc-analytics:govdoc-context]]
(справочник организаций молодёжной политики). **Не дублировать** —
указывать только ссылку.

## Архивация

Сущность с `last_referenced` > `thresholds.archive_entity_idle_months`
(default 6) → архив через [[cog-archivist]]. При повторном упоминании
разархивируется.

## Дубликаты и merging

- Поиск дубликатов через fuzzy-match (по имени, аббревиатуре,
  синонимам в [[05_global_glossary]]).
- При обнаружении дубликата — merge с приоритетом более полной записи.
- ID устаревшей записи остаётся в истории как alias.

## Команды

| Команда | Действие |
|---|---|
| `покажи сущности` | Все активные |
| `покажи сущности по проекту X` | Фильтр |
| `кто такой X` | Поиск person + fuzzy |
| `какая организация X` | Поиск org + fuzzy |
| `добавь сущность: <данные>` | Новая запись |

**Триггеры записи:**
- Упоминание в 2+ разных проектах.
- Явная команда `добавь сущность`.
- Импорт из [[govdoc-analytics:govdoc-context]] при активации.

**Триггеры чтения:**
- Reading ladder уровень 3 (предметная задача).
- Команда `кто такой X` / `какая организация X`.
- При создании официального письма — форма обращения.
