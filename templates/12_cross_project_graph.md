# 12 — Граф связей между проектами

Визуально-текстовое представление связей между проектами, сущностями,
инструментами, знаниями. Создаётся при обнаружении кластеров из **3+
проектов** (`thresholds.cluster_min_projects`).

**Разница с `_generated/graph.html`:**
- `_generated/graph.html` — автоматически собирается из всех карт и
  обновляется по post-write hook. Визуально интерактивный.
- `12_cross_project_graph` — **ручной слой осмысления**: куратор пишет
  текстом, что связывает проекты, почему это важно, какие выводы.

## Типы связей

| Тип связи | Что означает |
|---|---|
| `shared_entity` | Проекты имеют общих людей / организации |
| `shared_domain` | Проекты в одном домене |
| `shared_methodology` | Проекты используют одинаковые механизмы |
| `shared_artefact` | Проекты производят похожие документы |
| `sequential` | Один проект — продолжение другого |
| `parent_child` | Проект-зонтик и под-проекты |
| `conflict` | Проекты противоречат (аудит через [[14_audit_log]]) |

## Связь с _generated/graph.html

Ручные записи здесь **дополняют** автограф:
- Автограф показывает все связи (even weak ones).
- Этот файл — **кураторская интерпретация**: какие связи значимы и
  почему.
- При открытии `_generated/graph.html` кластеры этой карты
  подсвечиваются отдельным цветом.

## Правила записи кластера

1. **Порог:** минимум `thresholds.cluster_min_projects` проектов
   (default 3) с общей связью.
2. **Тип связи** указан.
3. **Инсайт обязателен** — зачем эта кластеризация нужна.
4. **ID уникальный** `cluster-NNN`.
5. **Верификация:** П13 «Аудитор».

## Правила записи инсайта

1. **Дата** обязательна.
2. **Ссылка на cluster-NNN** — откуда выведен.
3. **Действие** — что делать на основании вывода.
4. **Верификация П1** если вывод аналитический.

## Conflict-edges (противоречия)

Когда обнаруживается противоречие между проектами (один проект
декларирует X, другой Y) — создаётся conflict-edge + запись в
[[14_audit_log]]. В `_generated/graph.html` такие рёбра
подсвечиваются красным (v1.3).

## Команды

| Команда | Действие |
|---|---|
| `покажи граф` | Открыть `_generated/graph.html` |
| `покажи кластеры` | Активные кластеры |
| `покажи связи X` | Все связи проекта X |
| `покажи противоречия` | conflict-edges |
| `добавь инсайт` | Записать вывод по кластеру |
